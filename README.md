# Bank Customer Churn Analysis – Power BI Dashboard

## 🚀 Project Overview

Interactive Power BI dashboard exploring churn behaviour of 10K retail bank customers. It shows key KPIs and visualizes churn by geography, number of products and age group to highlight the riskiest customer segments.
The report is designed as a one‑page executive view 
---

## ✨ Key Features

Executive KPI summary with customer count, churned customers, churn rate %, average balance,
  active member % and average number of products.
- Breakdown of churn by geography, number of products and age group to quickly spot the
  riskiest customer segments.
- Comparison of average balance for active vs churned customers.
- Interactive filtering by geography, gender, credit card ownership and activity status to
  support ad‑hoc questions from business and risk teams.

---

## 🛠️ Tech Stack

- **Power BI Desktop** – data model, DAX measures, KPIs and interactive dashboard.
- **SQL** – exploration and validation of the source dataset before loading into Power BI.
- **Excel** – quick checks and calculations during exploratory analysis.
